Windows下强制格式化优盘

在Windows中，你可以使用命令提示符（CMD）来强制格式化U盘。以下是具体步骤：

  1. 打开命令提示符：

    • 按 Win + R，输入 cmd，然后按 Enter。

  2. 输入 diskpart 命令：

    • 在命令提示符窗口中输入 diskpart，然后按 Enter。这将打开 diskpart 工具。

  3. 列出所有磁盘：

    • 输入 list disk，然后按 Enter。这将列出所有连接到计算机的磁盘。

  4. 选择U盘：

    • 找到你的U盘对应的磁盘号（比如 Disk 1）。
    • 输入 select disk X（这里的 X 是你的U盘的磁盘号），然后按 Enter。

  5. 清除U盘：

    • 输入 clean，然后按 Enter。这将清除U盘上的所有数据。

  6. 创建新的分区：

    • 输入 create partition primary，然后按 Enter。这将创建一个新的主分区。

  7. 选择新分区：

    • 输入 select partition 1，然后按 Enter。这将选择新创建的分区。

  8. 格式化U盘：

    • 输入 format fs=ntfs quick，然后按 Enter。这将快速格式化U盘为NTFS文件系统。如果你想格式化为FAT32文件系统，可以将 ntfs 替换为 fat32。

  9. 分配驱动器号：

    • 输入 assign，然后按 Enter。这将分配一个驱动器号给U盘。

  10. **退出 diskpart**：

    • 输入 exit，然后按 Enter。这将退出 diskpart 工具。
